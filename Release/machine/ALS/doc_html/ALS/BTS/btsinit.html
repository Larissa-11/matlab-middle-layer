btsinit

PURPOSE

BTSINIT

SYNOPSIS

function btsinit(OperationalMode)

DESCRIPTION 

BTSINIT

CROSS-REFERENCE INFORMATION

This function calls:
  • at2bts AT2BTS - Converts simulator values to control system values
  • bts2at BTS2AT - Converts control system values to simulator values
  • getname_bts
  • gettv_bts [AM, tout, DataTime, ErrorFlag] = getx_bts(Family, Field, DeviceList, t, FreshDataFlag, TimeOutPeriod)
  • setoperationalmode SETOPERATIONALMODE - Switches between the various operational modes
  • settv_bts SETTV_BTS - Sets both the paddle and lamp
This function is called by:
  • aoinit AOINIT - Initialization function for the Matlab Middle Layer (MML)

SUBFUNCTIONS

  • function [Amps] = local_minsp(Family, List)
  • function [Amps] = local_maxsp(Family, List)
  • function tol = gettol(Family)
