arplot_orbitfb

PURPOSE

arplot_orbitfb(Month1 String, Days1, Year1, Month2 String, Days2, Year2)

SYNOPSIS

function arplot_orbitfb(monthStr, days, year1, month2Str, days2, year2)

DESCRIPTION 

 arplot_orbitfb(Month1 String, Days1, Year1, Month2 String, Days2, Year2)
 
 Plots archived data about the orbit feedback

 Example:  arplots_orbitfb('January',22:24, 1998);
           plots data from 1/22, 1/23, and 1/24 in 1998

 C. Steier, May 2002

CROSS-REFERENCE INFORMATION

This function calls:
  • arglobal Just used to make archived variables global
  • arread ARREAD - Loads one day of archived data into matlab global memory
  • arselect ARSELECT - Return the archived data for a group on channel names
