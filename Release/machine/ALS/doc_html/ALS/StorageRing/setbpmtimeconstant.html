setbpmtimeconstant

PURPOSE

SETBPMTIMECONSTANT - Sets the timeconstants for the Bergoz style BPMs.

SYNOPSIS

DESCRIPTION 

SETBPMTIMECONSTANT - Sets the timeconstants for the Bergoz style BPMs.
 setbpmtimeconstant(BPMTimeConstant, DeviceList)
 
  INPUTS
  1. BPMTimeConstant - Time constant [seconds]  {Default: .25}
