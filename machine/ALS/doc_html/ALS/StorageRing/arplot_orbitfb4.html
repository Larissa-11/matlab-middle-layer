arplot_orbitfb4

PURPOSE

arplot_orbitfb4(Month1 String, Days1, Year1, Month2 String, Days2, Year2)

SYNOPSIS

function arplot_orbitfb4(monthStr, days, year1, month2Str, days2, year2)

DESCRIPTION 

 arplot_orbitfb4(Month1 String, Days1, Year1, Month2 String, Days2, Year2)

 Plots archived data about the orbit feedback: plots the corrector
       trim AM values (e.g., SR04C___VCM1T__AM00)
       (this routine is compatible with the new Matlab ML)

 Example:  arplots_orbitfb4('January',22:24, 2007);
           plots data from 1/22, 1/23, and 1/24 in 2007

 C. Steier, May 2002; updated to work with new ML 4-25-07, T.Scarvie

CROSS-REFERENCE INFORMATION

This function calls:
  • arglobal Just used to make archived variables global
  • arread ARREAD - Loads one day of archived data into matlab global memory
  • arselect ARSELECT - Return the archived data for a group on channel names
  • switch2bergoz SWITCH2BERGOZ - Turn off the status of the non-Bergoz BPMs
