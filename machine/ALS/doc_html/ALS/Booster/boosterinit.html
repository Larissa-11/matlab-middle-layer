boosterinit

PURPOSE

BOOSTERINIT - Initialization function of the ALS booster

SYNOPSIS

function boosterinit(OperationalMode)

DESCRIPTION 

BOOSTERINIT - Initialization function of the ALS booster

CROSS-REFERENCE INFORMATION

This function calls:
  • at2booster AT2BOOSTER - Converts simulator values to control system values
  • booster2at BOOSTER2AT - Converts control system values to simulator values
  • getname_booster
  • setoperationalmode SETOPERATIONALMODE - Switches between the various operational modes
This function is called by:
  • aoinit AOINIT - Initialization function for the Matlab Middle Layer (MML)

  • function [Amps] = local_minsp(Family, List)
  • function [Amps] = local_maxsp(Family, List)
  • function tol = gettol(Family)
